CSS išdėstymo būdai

CSS išdėstymas yra vienas sudėtingiausiai perkandamų niuansų kuriant tinklapių dizainus. Nors jų rankinis kūrimas yra komplikuotas, egzistuoja jau sukurti sprendimai, kuriuos lengva perprasti ir kurie leidžia lengviau sukurti išdėstymus, kurie adaptuojasi prie skirtingų ekrano dydžių.

CSS išdėstymams kurti yra keletas skirtingų būdų, tačiau šiuo metu lanksčiausias ir populiariausias yra naudojant float savybę. Float priverčia elementą ištrūkti iš įprastos tvarkos ir lygiuotis į tam tikrą šoną. Tai galima išnaudoti net labai sudėtingų išdėstymų sukūrimui. Tokiems išdėstymams reikia panašaus kodo:

header {
  /* Viršaus bloko stilius */
}

#main {
  /* Apačios bloko stilius */
}

#main .left-sidebar {
  width: 200px;
  float: left;
}

#main .right-sidebar {
  width: 200px;
  float: right;
}

#content {
  margin: 0 200px 0 200px; /* top left bottom right */
}

/* #main elementas turi turėti clearfix klasę */
.clearfix:before, .clearfix:after {
  content: ' ';
  display: table;
}
.clearfix:after {
  clear: both;
}

footer {
  /* Apačios bloko stilius */
}

Kitas, labiau rekomenduojamas būdas sukurti išdėstymus yra naudojant kitų sukurtas sistemas. Jų populiariausia yra Bootstrap. Pačio Boostrap puslapyje yra labai daug instrukcijų ir pavyzdžių kaip sukurti išdėstymus, kurie tinkamai persirikiuotų prie skirtingų ekrano dydžių.
