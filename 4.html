CSS: aprašomasis dizainas

Ši paskaita yra susipažinimas su CSS - kalba, skirta aprašyti tinklapių dizainą žodžiais. Analogija čia panaši: kaip HTML'as atskleidžia tai, ką užslėptai turi Word'as, taip CSS atskleidžia tai, ką kažkur paslėptai laiko InDesign ar panašios programos. CSS'o esmė yra galimybė kodu įvardinti kokiems elementams priskirti kokį dizainą.

Pagrindinė informacija

CSS yra pakraunamas HTML'o, puslapio <head> elemente nurodant CSS adresą tokia forma:

<link rel="stylesheet" href="styles/style.css">

CSS taisyklės rašomos tokia forma:

selektorius, kitas selektorius {
  savybė: reikšmė;
  kita-savybė: jos-reikšmė;
}

Su selektoriais galima susipažinti žaidžiant CSS Diner žaidimą. Pagrindinės jų formos pristatomos iki 11 lygio, vėlesni lygiai yra sudėtingesni ir kartu galingesni būdai pasirinkti elementus.

CSS savybėmis galima nurodyti elementų šriftus, dydžius, spalvas, rėmus ir t.t. Žemiau yra daug įvairių taisyklių ir selektorių pavyzdžių su CSS komentarais.

p {
  font-family: Georgia, serif;
  font-size: 16px;
  font-weight: bold;
  line-height: 1.3;
  color: #ff9900;
  
  text-align: left;
  text-transform: uppercase;
  
  /*
  Padding - vidaus tarpai
  Margin - išorės tarpai
  */
  margin: 20px 0 16px 0;
  /*
  Tiek margin, tiek padding atstumai eina tokia tvarka:
  viršus dešinė apačia kairė
  (pagal laikrodžio rodyklę nuo viršaus)
  */
  padding: 1em 0 0.8em 0;
  /*
  CSS turi nemažai skirtingų matų, 
  kuriais galima nurodyti dydžius.
  Svarbiausi yra 2:
  px - pikseliai
  em - m raidės plotis
  */
  margin-top: 1.2em;
  padding-bottom: 1em;
  /*
  Padding ir margin galima nurodyti ir po vieną.
  Bendrai CSS turi nemažai tokių taisyklių,
  kurios į vieną bendrą sujungia keletą smulkesnių.
  */
  
  background: #efe;
  /* #efe = #eeffee, dažnas spalvos trumpinys */
  
  width: 200px;
  height: 40px;
  
  border-bottom: 1px solid #ccc;
}
