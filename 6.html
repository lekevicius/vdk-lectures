CSS šriftai, animacijos ir kitos galimybės

Paskutinė paskaita apie CSS skirta mažesnėms ir naujesnėms CSS savybėms: galimybei naudoti bet kokį šriftą, perėjimams ir animacijoms. Šios paskaitos tikslas kartu yra ir įtvirtinti CSS žinias, gebėjimą susigaudyti kode ir ištaisyti klaidas.

Pagrindinė informacija

CSS šriftų naudojimo kodas skiriasi priklausomai nuo to, kokį servisą naudojate. Kiekvienas jų turi savo instrukcijas, kurios dažniausiai labai paprastos, ir reikia pridėti eilutę arba dvi į HTML <head> elementą. Tuomet galima naudoti šriftą paprastai, pavyzdžiui:

p {
  font-family: 'lato', sans-serif;
}

CSS perėjimai yra paprasčiausias būdas pridėti dinamikos į puslapį. Juos labai paprasta aprašyti:

a {
  color: #ff9900;
  transition: all 0.3s;
}

a:hover {
  color: #ff0000;
}

Keli esminiai patarimai apie HTML ir CSS klaidų taisymą:

  • Įsitikinkite, kad redaguojate tą failą, kurį norite. Kartais galite reduoti kokį nors Untitled vietoje tikro failo.
  • Rašykite tiek HTML, tiek CSS kodą tvarkingai ir kruopščiai. Kartais net vienas praleistas simbolis ar nepadėtas tarpas gali sugriauti puslapį, todėl neskubėkite.
  • Web Inspector (tas, su kuriuo žaidėme pirmą paskaitą, ir kuris yra pasiekiamas per right-click "Inspect Element") yra geras įrankis paanalizuoti ne tik kitų, bet ir savo kodą, surasti klaidas.
